# 2026 예산 관리 시스템 | 1440 RESERVE

## 📋 프로젝트 개요

**Exclusive Budget Management Dashboard** - 교육문화팀의 2026년도 예산(총 10억원)을 효율적으로 관리하고 실시간으로 모니터링할 수 있는 프리미엄 웹 기반 대시보드입니다.

### 🎨 디자인 컨셉
고급스러운 다크 테마와 골드/실버 액센트를 활용한 럭셔리 디자인으로, 전문적이고 세련된 사용자 경험을 제공합니다.

### 프로젝트 목표
- 엑셀 기반 수작업 관리의 비효율성 해소
- 실시간 예산 현황 파악 및 투명한 히스토리 관리
- 예산 초과 사전 예방을 위한 알림 시스템 구축
- 통계 기반 예산 소진 예측 및 분석 기능 제공
- 집행 내역 수정 기능으로 유연한 데이터 관리
- 월별 예산 이동 추적으로 정밀한 예산 운영

---

## ✨ 주요 기능

### 1. 실시간 예산 현황 모니터링 ✅
- **3개 계정** (자체교육 4억, 위탁교육 3억, 컨소시엄 3억) 예산 현황 실시간 조회
- 계정별 **예산, 집행액, 잔액, 집행률** 카드형 대시보드
- 전체 요약 통계 (총 예산, 총 집행액, 총 잔액, 전체 집행률)
- 월별 배정 예산 대비 실제 집행 현황 상세 조회

### 2. 예산 집행 관리 ✅
- 예산 집행 내역 등록 (일자, 금액, 내용, 담당자, 이슈사항)
- **📝 집행 내역 수정 기능** (NEW) - 등록된 데이터 실시간 수정
- 계정별 필터링 기능으로 집행 내역 조회
- 집행 내역 삭제 기능
- 최신순 자동 정렬
- 수정/삭제 버튼으로 편리한 데이터 관리

### 3. 예산 이동 관리 및 히스토리 추적 ✅
- 계정 간 예산 이동 실시간 반영
- **📅 월별 예산 이동 추적** (NEW) - 출발 월과 도착 월 정보 관리
- 이동 전후 잔액 자동 기록 및 관리
- 이동 날짜, 사유, 금액, 월 정보를 포함한 완전한 히스토리 추적
- 계정별/월별/기간별 예산 이동 내역 조회
- 럭셔리 카드 UI로 이동 내역 가독성 향상

### 4. 스마트 알림 시스템 ✅
- **90% 도달 알림**: 계정별 예산 사용률 90% 도달 시 경고 팝업
- **100% 초과 알림**: 예산 100% 초과 시 긴급 경고 팝업
- 알림 히스토리 자동 저장 및 관리
- 시스템 자동 알림 로그 기록

### 5. 시각화 대시보드 ✅
- **🎨 다크 럭셔리 디자인** (NEW) - 고급스러운 골드/실버 액센트
- **계정별 막대그래프**: 예산, 집행액, 잔액 비교 차트 (다크 테마)
- **월별 집행 추이 그래프**: 1~12월 월별 집행 추이 선 그래프
- **월별 상세 현황 테이블**: 계정별 월별 배정 및 집행 현황 표
- 반응형 디자인으로 PC 환경 최적화
- Playfair Display 세리프 폰트로 우아한 타이포그래피

### 6. 통계 기반 예측 및 분석 ✅
- **예산 소진 예측**: 월평균 집행액 기반 예산 소진 시점 예측
- **트렌드 분석 차트**: 실제 집행 vs 예측 집행 비교 차트
- **통계 요약**: 집행률 최고 계정, 최대 단일 집행, 총 집행 건수 등
- 연말 예상 잔액 자동 계산

### 7. 데이터 내보내기 ✅
- 집행 내역 CSV 다운로드
- 예산 이동 내역 CSV 다운로드
- 월별 현황 리포트 CSV 다운로드
- 엑셀에서 바로 열 수 있는 한글 지원 (BOM 포함)

## 🚀 기능별 URI 및 사용법

### 메인 페이지
- **URI**: `/` 또는 `/index.html`
- **기능**: 대시보드 메인 화면

### 탭별 기능
1. **대시보드 탭**
   - 전체 예산 현황 요약 카드
   - 계정별 상세 현황 카드
   - 계정별 예산 집행 현황 차트
   - 월별 예산 집행 추이 차트
   - 월별 상세 현황 테이블

2. **집행 관리 탭**
   - 좌측: 예산 집행 등록 폼
   - 우측: 집행 내역 조회 및 관리
   - 계정별 필터링 지원

3. **예산 이동 탭**
   - 좌측: 예산 이동 등록 폼
   - 우측: 예산 이동 히스토리
   - 이동 전후 잔액 상세 표시

4. **알림 히스토리 탭**
   - 90% 도달 알림 내역
   - 100% 초과 알림 내역
   - 발생 일시 및 계정 정보

5. **예측 분석 탭**
   - 예산 소진 예측 카드 (계정별)
   - 월별 집행 트렌드 분석 차트
   - 통계 요약 카드

### RESTful API 엔드포인트
모든 데이터는 RESTful Table API를 통해 관리됩니다.

- **GET** `tables/budget_accounts` - 예산 계정 조회
- **GET** `tables/monthly_allocation` - 월별 배정 조회
- **GET** `tables/expenditures` - 집행 내역 조회
- **POST** `tables/expenditures` - 집행 내역 등록
- **DELETE** `tables/expenditures/{id}` - 집행 내역 삭제
- **GET** `tables/budget_transfers` - 예산 이동 조회
- **POST** `tables/budget_transfers` - 예산 이동 등록
- **GET** `tables/alert_logs` - 알림 로그 조회
- **POST** `tables/alert_logs` - 알림 생성

## 🚦 사용 가이드

### 1. 예산 집행 등록하기
1. 상단 네비게이션에서 **"EXPENDITURE"** 탭 클릭
2. 좌측 폼에서 정보 입력:
   - 계정 선택 (자체교육/위탁교육/컨소시엄)
   - 집행일자 선택
   - 집행금액 입력 (원 단위)
   - 내용/사유 작성
   - 담당자 이름 입력
   - 이슈사항 입력 (선택)
3. **"등록하기"** 버튼 클릭
4. 대시보드에 자동 반영 확인

### 1-1. 예산 집행 수정하기 ⭐NEW
1. 집행 내역 목록에서 수정할 항목 찾기
2. **파란색 수정 버튼 (✏️)** 클릭
3. 수정 모달에서 정보 변경
4. **"수정 완료"** 버튼 클릭
5. 변경사항 즉시 반영

### 2. 예산 이동하기 (월별 추적) ⭐UPDATED
1. 상단 네비게이션에서 **"TRANSFER"** 탭 클릭
2. 좌측 폼에서 정보 입력:
   - 출발 계정 선택
   - **출발 월 선택** (1~12월) ⭐NEW
   - 도착 계정 선택
   - **도착 월 선택** (1~12월) ⭐NEW
   - 이동금액 입력
   - 이동 사유 작성
3. **"이동 실행"** 버튼 클릭
4. 이동 전후 잔액 자동 기록
5. 우측에서 월별 이동 히스토리 확인
6. 럭셔리 카드 UI로 이동 내역 상세 표시

### 3. 알림 확인하기
- 예산 사용률이 90% 이상이 되면 우측 상단에 자동 알림 팝업
- **"알림 히스토리"** 탭에서 과거 알림 내역 조회 가능

### 4. 데이터 내보내기
1. 우측 상단 **"내보내기"** 버튼 클릭
2. 원하는 데이터 유형 선택:
   - 집행 내역 다운로드
   - 예산 이동 내역 다운로드
   - 월별 현황 다운로드
3. CSV 파일 다운로드 완료
4. 엑셀에서 파일 열기 (한글 자동 인식)

### 5. 예측 분석 보기
1. **"예측 분석"** 탭 클릭
2. 계정별 예산 소진 예측 확인

## 🔒 데이터 보안 및 권한

### 현재 구현 상태
- 전체 사용자가 모든 데이터 조회 및 수정 가능
- 별도의 인증/인가 시스템 없음
- 데이터는 RESTful Table API에 저장

### 권장사항
- 실제 운영 시 사용자 인증 시스템 도입 권장
- 예산 이동 등 중요 작업에 승인 프로세스 추가 권장
- 데이터 백업 정책 수립 권장

## 🐛 알려진 제한사항

1. **동시 접속 제한**: 5명 이내 동시 접속 권장
